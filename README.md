# Frosted by Shristi 🍰

Frosted by Shristi is a modern bakery website developed using HTML, CSS, and JavaScript. The website showcases a variety of cakes, cheesecakes, and sweet treats through an attractive and responsive user interface.

## Features

* Responsive design for desktop and mobile devices
* Dark mode toggle
* Cake category filtering
* Smooth scrolling navigation
* Interactive hover animations
* Scroll-to-top button
* Online order form
* Multiple cake categories and special editions

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Purpose

This project was created to practice frontend web development concepts, including responsive layouts, DOM manipulation, animations, and user interface design.
